##  About the Project

**Araba İlanları** is a fully responsive car sales and rental platform showcasing a premium dark-themed design. This project demonstrates modern web development practices including semantic HTML5 structure, CSS custom properties for maintainability, and Bootstrap 5 for responsive grid layouts.

The interface features an intuitive search experience, categorized vehicle listings, and a professional card-based design optimized for both mobile and desktop users.

---

##  Key Features

- ** Premium Dark Theme** - Modern, eye-catching design with golden accents
- ** Fully Responsive** - Optimized for Mobile, Tablet, and Desktop viewports
- ** Interactive Search** - Brand, category, and model filtering system
- ** Card-Based Listings** - Elegant car cards with hover animations
- ** Accessible Design** - Semantic HTML5 and ARIA attributes
- ** Performance Optimized** - CSS custom properties and efficient selectors

##  Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, Safari)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/araba-ilanlari.git
   ```

2. **Navigate to the project**
   ```bash
   cd araba-ilanlari
   ```

3. **Open in browser**
   - Double-click `index.html`, or
   - Use Live Server extension in VS Code
